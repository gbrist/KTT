Kendall Toy Test App
----------------------

This app allows you to administer and score the Kendall Toy Test with customizable inputs.

Features:
- Select correct items by clicking images.
- Specify environment (Quiet/Noise), aided status, and presentation/noise level.
- Calculate and display the score out of any number of presented items.
- Download results as a PDF or PDF with graph.

Setup:
1. Open index.html in any modern browser.
2. Place all stimulus images in the 'images' folder (already created).
3. Ensure filenames match those listed in the script.

Image files expected:
- tree.png
- car.png
- Star.jpg
- mouse.png
- duck.png
- horse.png
- truck.jpg
- fork.png
- sheep.png
- cow.png
- house.png
- ball.png
- cup.png
- bath.png
- key.png

Output:
- PDF results including metadata and performance summary.
- Optional bar graph in exported report.

Developed for: Speech & Language / Audiology assessment use.
